# Sistema de Cadastro de Motoristas e Veículos

Este é um sistema simples para cadastro de motoristas e veículos, desenvolvido em React.

## Como Usar

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/nome-do-repositorio.git


2 **Instale as Dependências::**

Navegue até o diretório do projeto e execute o seguinte comando para instalar as dependências:

npm install


3 **Execute o Projeto:**

Após a instalação das dependências, você pode iniciar o projeto com o comando:

npm start

Isso iniciará o servidor de desenvolvimento e abrirá a aplicação no seu navegador padrão.

4 **Acesse a Aplicação::**

A aplicação estará disponível em http://localhost:3000 no seu navegador.

Funcionalidades
-Cadastro de motoristas com nome, documento e veículo associado (opcional).
-Cadastro de veículos com marca, modelo e placa.
-Edição e exclusão de motoristas e veículos.
-Visualização dos motoristas e veículos cadastrados.


Tecnologias Utilizadas
-React
-React Router Dom
-TypeScript
-React Toastify

Estrutura do Projeto
-src/components: Contém os componentes React reutilizáveis.
-src/pages: Contém as páginas da aplicação, cada uma representando uma rota específica.
-src/types: Define os tipos de dados utilizados na aplicação.
-src/img: Armazena as imagens utilizadas na aplicação.
